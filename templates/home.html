Do Women Earn 81¢ For Every Dollar Men Earn?

Nationwide, amongst people working full time, women earn 81 cents for every dollar that men earn. This wage gap has been persistent over time and does not appear to be narrowing significantly. This project therefore seeks to explore the differences in earnings between men and women across different occupations; starting from choices in college major.

See the Research Question page for more information about why this topic is important.

The data collection and analysis program is described in the Project Description page.

If you would like to learn more about the research flow, see the Methodology page.
